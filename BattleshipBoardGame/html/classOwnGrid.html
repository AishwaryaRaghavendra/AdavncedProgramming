Detailed Description

Class OwnGrid has its grid of rows and columns. Holds the vector of ships. Holds the map of ship's length and no. of ships. Places the ship in its grid. Takes the blow and returns the Impact for that blow.

Shot::Impact OwnGrid::takeBlow ( const Shot &  shot)

Method takeBlow() defined takes the shot at targetPosition. checks if the targetPosition is in the occupied area of the ship. checks if there is a Impact of HIT, NONE or SUNK on the ship of the OwnGrid and returns that Impact.

Parameters
shotobject of Shot class passed as reference argument. Sets the shot at targetPosition.

Steps: First insert the shot of desired targetPosition into shotAt set

Iterate over all ship's to find the hit, miss or sunk

Iterate over the ship's occupiedArea which is set of gridPosition

To check if Impact of the shot after takeBlow is a Hit by comparing targetPosition and ship's occupied gridPosition

If the size of the intersection is one less than the length of the ship, the current hit is the final one and ship is sunk

Return shot Impact is SUNKEN

Return shot Impact is HIT

Else return shot Impact is NONE
