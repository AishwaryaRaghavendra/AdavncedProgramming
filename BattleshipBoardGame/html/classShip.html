Detailed Description

Class Ship defined for ship and has two grid positions namely bow and stern. Checks if the grid positions for bow and stern are valid. Returns bow, stern and length of the Ship. Calculates the occupied area and blocked area for the ship and returns the occupied area and blocked area as set of grid positions.
