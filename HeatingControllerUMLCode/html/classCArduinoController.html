Detailed Description

Class CArduinoController employs Arduino Board as the Controller Unit for the Process. Power Supply is ON for it to function else the Controller is in OFF state Performs the Analog to Digit Conversion of the Voltage received from the Sensor. Performs degrees Celsius Conversion. Executes the Comparator Logic that is to check the difference between the Sensor Temperature and desired(User Input)temperature where the error signal = UserTemp - SensorTemp(Feedback) If the error signal >0, Heater Turn ON and if error signal <0,Cooler Turn ON. Sets the appropriate Relay Drivers ON of either Heater or Cooler depending on the Comparator Logic
